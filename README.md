# 📱 CSS Media Queries Responsive Layout

A simple responsive web page built using **HTML5** and **CSS3** to demonstrate the use of **CSS Media Queries**. The project contains multiple content boxes with paragraph text that automatically adjust their layout based on the screen size, providing a better user experience on desktops, tablets, and mobile devices.

---

## 🚀 Features

- Responsive design using CSS Media Queries
- Flexbox layout
- Automatically adjusts for different screen sizes
- Clean and minimal user interface
- Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Media Queries

---

## 📱 Responsive Behavior

- 💻 Desktop: Content boxes are displayed in a row.
- 📱 Mobile: Content boxes are stacked vertically using CSS Media Queries.

---

## 🎯 Learning Outcomes

- Understanding CSS Media Queries
- Creating responsive web layouts
- Using Flexbox for alignment
- Building mobile-friendly webpages
